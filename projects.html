RePoste

RePoste is an open source replay system for the sport of Fencing. The application allows referees to replay any point, missed hit, or foul to ensure a fair match. The RePoste system applies simple controls and UI so anyone can easily use the application during an event or match.

ChatMaps

A social networking application that allows users around the same location to create or connect to chat rooms and engage with each other on related topics. The app displays a map of the users general location to see active chat rooms in their area while staying anonymous as all chatters locations are randomized to prevent tracking. ChatMaps is a safe and secure way to connect with others around you.
